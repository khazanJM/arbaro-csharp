Welcome to the Arbaro CSharp GitHub Pages.

This project is an effort to port the Arbaro project to C Sharp. Arbaro is a botanical 3D tree generator based on the seminal paper from Weber & Penn: Creation and rendering of realistic trees.

Status

The current effort is to port Arbaro from Java to C-Sharp and to provide a DirectX viewer of the generated tree. Nothing really working at the moment, please check at a later stage :-)

The original Arbaro project lives here. The Direct X implementation I'm creating is more or less derived from the Rastertek tutorials you can find here.
